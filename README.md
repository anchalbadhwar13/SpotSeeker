# SpotSeeker - AI Parking Spot Finder
# Urban drivers waste significant time searching for parking, especially during peak hours. This project builds an AI‑powered system that predicts parking availability at a given time and location using supervised machine learning models.
# The system uses features such as hour of day, day of week, weather, and historical occupancy patterns to classify parking availability into categories (Empty, Moderate, Busy, Full) or provide a probability score.

# We develop a supervised machine learning system that:
# - Takes structured inputs (time, weather, location, etc.)
# - Learns from historical or synthetic parking data
# - Predicts availability as a class or probability
# - Provides interpretable outputs and visualizations

# Inputs
# - Hour of day
# - Day of week
# - Weather
# - Location ID
# - Historical occupancy
# - Special event flags
# Outputs
# - Availability class (Empty → Full)
# - Probability scores
# - Optional binary: Available / Not Available
